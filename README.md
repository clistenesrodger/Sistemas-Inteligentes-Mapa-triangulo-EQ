# Gerador de Mapas com Triângulos - Versão para Grandes Volumes

## Sobre o Programa

Gera mapas 2D com triângulos equiláteros sem colisão. Otimizado para:
- 300+ triângulos
- Execução rápida
- Validação automática
- Avisos quando não for possível gerar todos
