1. Descrição do Problema

Este projeto tem como objetivo a construção de um mapa bidimensional contendo obstáculos no formato de triângulos equiláteros, onde deve ser possível encontrar uma rota do ponto inicial até um ponto final utilizando apenas vértices visíveis dos obstáculos.

Os triângulos representam obstáculos no ambiente, portanto não pode haver colisão entre eles, nem sobreposição entre suas áreas. Após a geração do mapa, deverá ser calculado um caminho válido entre dois pontos, respeitando as restrições impostas pelos obstáculos.

O planejamento da rota deve considerar apenas linhas de visão entre vértices visíveis, evitando que o caminho atravesse qualquer triângulo.
