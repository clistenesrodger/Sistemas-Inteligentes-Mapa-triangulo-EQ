## 📸 Para Adicionar Screenshots (Recomendado)

Execute alguns exemplos e capture screenshots:

```bash
# Exemplo 1: Mapa pequeno (rápido)
python main.py --largura 500 --altura 500 --qtd 50 --lado 20 --xf 450 --yf 450

# Exemplo 2: Mapa complexo com retas
python main.py --largura 800 --altura 600 --qtd 100 --lado 18 --xf 760 --yf 560 --plotar-retas
```

Coloque os screenshots em uma pasta `screenshots/` e adicione ao README:
